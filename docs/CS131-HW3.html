Business Use Case Name: International flight check-in

Trigger: Customer approaches check in

Preconditions: Ticket, e-ticket, name and flight number, at least destination

Post conditions: Boarding pass, some helpful instructions

Actors: Check-in agent, client, computer system, airline company

Basic flow:

  1. Does the passenger have a ticket?
  2. If not, do they an e-ticket?
  3. If yes, do they know the booking record locator number? If not, do they know the flight number? If not, do they know the destination?
  4. Begin checking process
  5. Request passport, check photo and expiration date
  6. Does the booking have a frequent flyer (FF) number attached?
  7. If not, do they want to join?
  8. If yes, attach temporary FF number to booking so they can still get the benefits of the flight.
  9. Does the booking already have an assigned seat?
  10. If not, find an acceptable seat for the passenger, ask if they prefer aisle or window.
  11. If yes, ensure that the seat is okay with the passenger.
  12. Print boarding passes
  13. If the passenger is checking bags, print tags and send bags down conveyor belt
  14. Check if the passenger's carry on is of acceptable size and weight
  15. Ensure passenger knows where they're headed, and what time their flight leaves
