MATH31-comparison-tests

Comparison tests

Idea: Compare one series to another with a known convergence/divergence (geometric, harmonic, p-series, etc).

Suppose we have two series ∑an and ∑bn with positive terms:

  1. If ab<bn for all n, and ∑bn converges, then ∑an also converges.
  2. If ab>bn for all n, and ∑bn diverges, then ∑an also diverges.

If you show divergence for bn when an<bn, it proves nothing. The upper series diverges up to infinity and that tells us nothing about the lower series. It may diverge or converge. So make sure to show the correct comparison. (The same useless comparison is showing convergence for bn when an>bn).

Example 1

∑n=1∞1n2+2

First check if it fails divergence test, look for other known series, see if integral test could work (it would), but theres a better way. Consider this comparison:

0≤1n2+2≤1n2

If we're trying to show convergence, we need this to be less than something we know convergence for.

So lets show convergence of 1n2:

∑n=1∞1n2

This is a P-series with p=2, and since our p>1 it means the series converges. This means by the comparison test that the original problem also converges.

Example 2

∑n=1∞13+2n

All the terms are positive on the interval, consider this comparison:

0≤13+2n≤12n

So try to determine convergence of the rightmost fraction:

∑n=1∞12n=∑n=1∞(12)n=∑n=1∞(12)(12)n−1

This is a geometric sum with a=12 and r=12, therefore since r<1 this will converge. So by the comparison test the original problem also converges.

Example 3

∑n=3∞3n2n−4

All the terms on the interval are positive, so lets try a comparison:

3n2n−4≥3n2n

We're trying to show divergence of the rightmost fraction, and that will show that the original also diverges.

∑n=3∞3n2n=∑n=3∞(32)n=∑n=3∞(32)3(32)n−3

Since this is a geometric and our r>1, it diverges. Therefore the original problem diverges.

Example 4

∑n=1∞1n‾√+1

Consider this comparison:

0≤1n‾√+1≤1n‾√

This is a P-series with p=12 (less than 1). This means it diverges. This shows nothing about the original problem!

When you can't use a comparison like this, you can use the limit comparison test.

Limit comparison test

Idea: If ∑an and ∑bn have positive terms, and this limit exists:

limn→∞anbn

If this limit exists, then that means that both terms are so close together that it doesn't matter. That means that both series either converge or diverge.

If it goes to infinity then the terms must difference enough that one or both of them diverges.

Proof

Suppose this limit exists:

limn→∞anbn=L

Then by definition:

∣∣∣abbn−L∣∣∣<ϵ

So:

−ϵL<abbn−L<ϵLL−ϵL<abbn<L+ϵL(1−ϵ)L⋅bn<an<(1+ϵ)L⋅bn

(1+ϵ)L is just a constant (doesn't affect the convergence/divergence of the series), so if bn converges an is less than that so it also converges. If bn diverges, then an is greater than that so it also diverges.

Example 5

So to take a look again at Example 4 above:

∑n=1∞1n‾√+1

Lets try the limit comparison test where:

limn→∞anbn

and

bn=1n‾√

So:

limn→∞1n√+11n√=limn→∞1n‾√+1⋅n‾√1=limn→∞n‾√n‾√+1=limn→∞11+1n√=1

Since our limit exists, an and bn are so close together, that if one converges the other must also. If one diverges the other must also. Just because the limit exists it doesn't mean they converge! They will just have the same result.

Now we know that bn diverges (p-series with p<1), it means the an does also!

Example 6

∑n=1∞2n2+n4n7+3‾‾‾‾‾‾‾‾√

Lets choose a bn that we know convergence/divergence. Start by trying a bn that models the end behavior of an:

∑n=1∞2n24n7‾‾‾‾√=∑n=1∞1n3/2

This is a P-series with a p=32, since p>1 it converges!

Limit comparison test:

limn→∞anbn=limn→∞2n2+n4n7+3√1n3/2=limn→∞2n2+n4n7+3‾‾‾‾‾‾‾‾√⋅n3/2=limn→∞2n7/2+n5/24n7+3‾‾‾‾‾‾‾‾√=limn→∞2+1n4+3n7‾‾‾‾‾‾‾√=1

Since we know the limit exists, and we know that bn converges, an must converge also!

Example 7

∑n=1∞n‾√+lnnn2+1

Lets use the limit comparison test, and compare to bn of:

bn=∑n=1∞n‾√n2=∑n=1∞1n3/2

This is a P-series with p=32, so bn converges (p>1).

Limit comparison test:

limn→∞n‾√+lnnn2+1⋅n3/2∴=limn→∞n2+n3/2lnnn2+1=limn→∞1+lnnn1/21+1n2=limn→∞1+01+0=1=limn→∞lnnn1/2Use L'Hospitals=limn→∞1n12n√=limn→∞2n‾√nUse L'Hospitals=limn→∞2n‾√=0

So since the series bn converges, the series an also converges.
